# Data Directories

This repository keeps the code structure for data processing, but does not commit raw inputs or generated outputs.

- `raw/`: place monthly arXiv JSONL inputs here
- `output/`: generated monthly pipeline outputs
- `recent.jsonl`: generated by the daily update script
- `weekly/`: generated weekly trend reports
- `analysis/`: generated daily analysis snapshots

Use `make pipeline`, `python pipeline/daily_update.py`, and `python pipeline/weekly_trend.py` to regenerate these files locally.
